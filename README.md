> 第一步：bower install

> 第二步：npm install 或者 yarn install

> 第三步：gulp

- gulp dev : 自動開啟瀏覽器預覽
- gulp clean/gulp cleantmp : 清除一些暫時性檔案
- gulp deploy: 上傳到github (棄用，懶得修了XD)
    - 目前Deploy改使用Travis
